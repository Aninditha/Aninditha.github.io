LZW ENCRYPTION ALGORITHM


Duration: March 2016

Language & IDE: Java & Eclipse/Netbeans


Implementation details:
  • Implemented a java algorithm to compress large files.
  • Implemented an encoding algorithm that reads an input sequence of characters and converts them into integer codes. It then converts them into bytes codes.
  • Implemented a decoding algorithm that reads the output produced by the encoding algorithm and converts them back to the strings they represent by using the compression table made while ecoding.
  • Achieved a compression ratio of 0.38 for a sample dataset. (This ratio varies based on the dataset used for testing. Higher the redundancy, lower the ratio).
About LZW: The Lempel–Ziv–Welch (LZW) algorithm is a lossless data compression algorithm. LZW is an adaptive compression algorithm that does not assume prior knowledge of the input data distribution. This algorithm works well when the input data is sufficiently large and there is redundancy in the data. Two examples of commonly used file formats that use LZW compression are the GIF image format served from websites and the TIFF image format. LZW compression is also suitable for compressing text files, and is the algorithm in the compress Unix file compression utility.
